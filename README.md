# SAFE FORTNITE SKINCHECKER
This version does NOT use login, OAuth, tokens, device auth or credentials.

## How users load their locker:
1. Go to FortniteTracker.com → Locker → Export JSON
2. Upload the JSON file to your site
3. Your backend parses the cosmetics and returns:
   - skins
   - backblings
   - pickaxes
   - emotes
   - exclusives

This method is 100% safe and allowed.
